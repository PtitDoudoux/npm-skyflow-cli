<h1 align="center">Shell documentation</h1>

The **Shell** system allows you to use the skyflow commands in context.


## Usage

_**Enter in shell mode**_

```
skyflow shell docker
```

_**Exit shell mode**_

```
skyflow shell -e
```
or

```
skyflow shell --exit
```

_**Check the shell in use**_


```
skyflow shell
```
